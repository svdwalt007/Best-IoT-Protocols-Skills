
# Smart Home & Consumer IoT

## Matter 1.x
- CSA standard
- PASE commissioning via BLE discriminator + SPAKE2+
- CASE operational via NOC/ICAC/RCAC certificates
- Fabric model, clusters, device types (light, switch, sensor, lock, thermostat)
- Matter 1.3 additions: EV charger, water heater, microwave
- Controller/bridge model
- Matter SDK / matter.js

## HomeKit
- HAP-IP (HAP over IP using HTTPS+mTLS)
- HAP-BLE
- HAP pairing SRP method
- Accessory categories

## Google Home
- Google Home platform
- Local execution
- Google Assistant integration

## Amazon Sidewalk
- LoRa 900MHz sub-GHz / FSK / BLE
- Amazon Sidewalk Bridge devices
- Sidewalk endpoints, coverage across US

## Alexa Connect Kit (ACK)
- ACK SDK
- ACK device types
- Alexa provisioning via app

## DECT
- DECT ULE (Ultra Low Energy, 1.9GHz, medical alert/home automation)
- DECT NR+ (ETSI DECT-2020, unlicensed 1.9GHz sub-6GHz, private network IoT)
