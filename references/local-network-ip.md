
# Local Network & IP Protocols

## Wi-Fi
- Generations: 802.11 a/b/g/n/ac/ax/be (speeds, MIMO, MU-MIMO, OFDMA)
- Wi-Fi HaLow 802.11ah (sub-1GHz, IoT range, TWT)
- Wi-Fi Aware / NAN (Neighbor Awareness Networking)

## Ethernet
- 802.3 variants
- IEEE 802.1Q VLANs
- 802.1X (port-based NAC, EAP-TLS, EAP-PWD)

## 6LoWPAN & RPL
- RFC 4944 / 6282 (header compression, fragmentation, mesh addressing)
- RPL RFC 6550 (DAG, DODAG, DIO/DAO/DIS)

## IPv4 / IPv6
- DHCPv4 / DHCPv6, SLAAC, NDP, prefix delegation
- Transition mechanisms: NAT64/DNS64 (RFC 6146/6147), MAP-T (RFC 7599), MAP-E (RFC 7597)

## Discovery Protocols
- mDNS RFC 6762 / DNS-SD RFC 6763
- Used for Matter device discovery, HomeKit, AirPlay

## QUIC
- RFC 9000
- 0-RTT, connection migration, TLS 1.3 integrated, HTTP/3
