
# Edge Computing & Gateways

## MQTT Sparkplug B
- spBv1.0 namespace
- NBIRTH / DBIRTH / NDATA / DDATA / NCMD / DCMD
- Protobuf payload
- Primary Host concept
- Sparkplug 3.0 (Eclipse Foundation)

## LwM2M Gateway
- LwM2M v1.2 §9 / v2.0 extended proxy
- Cross-ref LwM2M skill

## Edge Platforms
- EdgeX Foundry (microservices: device service, core data, core command, support-notifications, app service)
- Azure IoT Edge (IoT Edge Hub, module instances, OPC Publisher connector, Modbus adapter, BLE adapter, IoT Central integration)
- AWS IoT Greengrass (Greengrass Core, Lambda deployment, Greengrass Stream Manager, AWS IoT SiteWise for industrial data)

## Industrial Adapters
- OPC UA PubSub (TSN/MQTT/AMQP transport bindings, UADP encoding, SecurityKeyService)
- Eclipse Hono (protocol adapters: MQTT/AMQP/HTTP/CoAP, AMQP 1.0 northbound Messaging Network, Command & Control bidirectional)
- KNX IoT (KNX-IoT-TP: CoAP+OSCORE over IPv6, bridges KNX TP/RF to IP/cloud)
