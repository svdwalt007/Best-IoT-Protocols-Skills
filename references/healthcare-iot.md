
# Healthcare IoT

## IEEE 11073 PHD (Personal Health Devices)
- 10404 Pulse Oximeter, 10406 Basic ECG, 10407 Blood Pressure monitor, 10415 Weight Scale, 10417 Glucose Meter
- 20601 Optimised Exchange Protocol (OEP)
- 10103 Nomenclature/coding

## Bluetooth SIG Health Profiles
- Health Device Profile (HDP)
- Continuous Glucose Monitor GATT profile
- Blood Pressure GATT profile
- Heart Rate GATT profile

## Health Informatics Standards
- HL7 FHIR R4/R5 (REST/JSON/XML)
- Resource types: Device, DeviceMetric, Observation, Patient
- SMART on FHIR OAuth2 authorisation
- FHIR Bulk Data API for IoT telemetry

## Integration & Testing
- IHE PCD (Patient Care Device, integration profiles: DEC, ACM, PIV)
- POCT1-A2 (IHE PCD / ISO 11073, point-of-care testing)

## Regulatory Context
- FDA 21 CFR Part 11 (electronic records/signatures)
- IEC 62304 (medical device software lifecycle, safety classes A/B/C)
- IEC 62443 cybersecurity for medical
- EU MDR 2017/745 (Medical Device Regulation, CE marking, UDI)
- HIPAA for US healthcare IoT data
