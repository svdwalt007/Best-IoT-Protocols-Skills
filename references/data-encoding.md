
# Data Encoding & Modeling

## Binary & Text Formats
- CBOR RFC 8949 (deterministic encoding, tags, CBOR sequences)
- SenML RFC 8428 (records: time/name/unit/value, pack, SenML-CBOR CF 112, SenML-JSON CF 110)
- JSON / JSON-LD / JSON Schema
- Protocol Buffers v2/v3 (proto3 syntax, well-known types, gRPC)
- FlatBuffers / MessagePack / BSON / Apache Avro

## YANG Models
- YANG RFC 7950 (module structure, types, YANG Catalog, YANG 1.1 action/anydata)
- NETCONF Datastores NMDA RFC 8342 (candidate/running/startup/operational)
- YANG Push RFC 8641 (configured/dynamic subscriptions, on-change/periodic)

## Industrial & Application Models
- OPC UA information model (XML/binary/JSON encoding, NodeID types, namespace)
- FIWARE NGSI-LD (JSON-LD @context, Entity/Attribute/Relationship, Context Broker)
- Haystack 4 (Axon, Zinc/JSON/Trio, tags, dicts, grids)
- SCHC RFC 8724 (context compression, fragmentation modes)
- IEEE 1451 / TEDS (self-describing sensor metadata, TEDS format, IEEE 1451.5 wireless)
- IPSO Smart Objects / OMNA Registry (resource 5700=SensorValue, 5701=SensorUnits, standard resource reuse)
