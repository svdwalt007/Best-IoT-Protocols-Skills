
# PAN & Short Range Protocols

## Bluetooth
- Classic BR/EDR
- BLE GAP/GATT/ATT/SMP architecture
- BT Mesh v1.0/1.1 (publish-subscribe, friendship, proxy)
- BT 5.x features: Long Range Coded PHY, High Speed 2M PHY, AoA/AoD
- LE Audio / LC3 / Auracast

## Zigbee
- IEEE 802.15.4 + ZHA + PRO
- Coordinator / Router / End-device
- Binding

## Z-Wave
- 500/700/800 series
- S2 security
- SmartStart
- IP licensing note

## Thread / OpenThread
- Mesh with border router
- Commissioner, joiner, PSKC

## Matter 1.x
- PASE commissioning via BLE
- CASE operational
- Clusters, device types, fabric
- Matter 1.3: EV charger, water heater, microwave

## IEEE 802.15.4
- MAC security (AES-128 CCM*, key management, frame counter)
- 6TiSCH (RFC 8180/9030/8607) - 6top, TSCH schedule, RPL routing

## Industrial Wireless
- WirelessHART (IEC 62591)
- ISA100.11a (IEC 62734)

## UWB (Ultra-Wideband)
- IEEE 802.15.4a/4z (ranging)
- CCC Digital Key 3.0

## Other
- DECT NR+ / DECT ULE
- EnOcean
- NFC / RFID
