
# Automotive & Transport Protocols

## Vehicle Diagnostics
- OBD-II / ISO 15765-4 (CAN-based on-board diagnostics, J1962 connector, PIDs, Mode 01-10, DTC codes)
- UDS ISO 14229 (Unified Diagnostic Services: 14229-1 base, 14229-3 DoIP over Ethernet Rel-2, 14229-5 CAN transport)
  - Services: 0x10 DiagnosticSessionControl, 0x27 SecurityAccess, 0x34/36/37 FOTA transfer, 0x31 RoutineControl
- SAE J1939 (CAN-based heavy vehicle protocol)

## Vehicle Architecture
- AUTOSAR Classic (layered ECU software architecture, RTE, COM/DCMM/UCM stacks)
- AUTOSAR Adaptive (POSIX, service-oriented, Ethernet, SOME/IP, ara::com, SDV)

## Connectivity & Keys
- V2X cross-ref (IEEE 802.11p, C-V2X)
- CCC Digital Key 3.0 (Phase 2 NFC + Phase 3 UWB precise ranging, Apple Wallet/Android car keys, IEEE 802.15.4a ranging)

## EV Charging
- ISO 15118 EV charging communication
  - 15118-2: AC/DC HomePlug GreenPHY, PLC transport
  - 15118-20: bidirectional V2G + wireless WPT
  - Plug & Charge (PnC): X.509 certificates, EVCC/SECC
- OCPP (Open Charge Point Protocol OCA)
  - OCPP 1.6 JSON/SOAP WebSocket
  - OCPP 2.0.1: device model, smart charging, security, firmware update
  - OCPI roaming protocol

## Cybersecurity & OTA
- UNECE WP.29 R155 cybersecurity management (CSMS for vehicles, TARA, type approval)
- UNECE WP.29 R156 software update management (SUMS, OTA update logging, type approval)
