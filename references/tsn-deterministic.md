
# TSN & Deterministic Networking

## TSN Overview
- IEEE 802.1AS (time sync ±10μs over Ethernet)
- 802.1Qbv (scheduled traffic / time-aware shaper)
- 802.1Qav (credit-based shaper)
- 802.1CB (frame replication/elimination for 99.9999% reliability)
- 802.1Qcc (centralized/distributed configuration)
- 802.1Qdd (resource allocation)

## PTP (Precision Time Protocol)
- IEEE 1588 v2 / v2.1
- Ordinary / boundary / transparent clocks
- Profiles: default, power, telecom G.8265.1, automotive
- Hardware timestamping
- Grandmaster election (BMCA)

## DetNet
- RFC 8655 architecture / RFC 8938 data plane
- MPLS+IP data planes
- Guaranteed bandwidth, latency, reliability

## TSCH & 6TiSCH
- TSCH IEEE 802.15.4e (time slots, channel hopping, slotframes)
- 6TiSCH RFC 8180 / 9030 (full stack: 6top RFC 8607, minimal schedule, PCE-based scheduling, RPL routing over TSCH)
