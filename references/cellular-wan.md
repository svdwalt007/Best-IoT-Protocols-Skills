
# Cellular WAN Protocols

## 2G / GPRS / EDGE
- GSM data layer
- GPRS classes

## 3G / HSPA / WCDMA
- UMTS architecture

## 4G LTE
- LTE-Advanced
- Cat-1/4/12
- NB-IoT / LTE-M (Cat-M1)

## 5G NR
- NSA/SA modes
- Network Slicing
- URLLC (sub-1ms)
- mMTC (1M devices/km²)
- 5G LAN-type service
- 5G NIDD/NEF

## EC-GSM-IoT
- 33dB MCL improvement

## 3GPP SCEF / NEF
- Rel-13 SCEF / Rel-15+ NEF
- T8 interface
- NIDD API
- Device Triggering (TS 23.682)
- Monitoring Events (reachability, location, loss of connectivity)

## V2X Protocols
- IEEE 802.11p / WAVE DSRC 5.9GHz
- C-V2X PC5 sidelink + Uu interface
- NR-V2X Rel-16
- SAE J2735 BSM/SPaT/MAP
- SAE J2945

## Satellite IoT / NTN
- Iridium SBD
- Inmarsat BGAN M2M
- 3GPP NTN Rel-17 direct-to-LEO NB-IoT/LTE-M
